# Boilerplate-.NET
Boilerplate para .net utilizando Clean Architecture com Vertical Slices, CQRS, Mediator e Unit Of Work Pattern.

É overengineering olhando puramente o repositório, mas a ideia é ser uma base para projetos maiores.

Tem apenas uma Command e uma Query de exemplos, utilizando as melhores e mais atuais práticas do mercado

Estamos utilizando EF Core para persistência de dados, com IAuditable para salvar dados de auditoria automatico.

Tratamento de exceçoes moderno com IExceptionHandler
